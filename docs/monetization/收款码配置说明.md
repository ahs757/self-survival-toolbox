# 收款码配置说明

## 快速配置步骤

### 1. 获取收款码图片
- **微信收款码**: 打开微信 → 我 → 服务 → 收付款 → 二维码收款 → 保存收款码
- **支付宝收款码**: 打开支付宝 → 首页 → 收钱 → 保存收款码

### 2. 上传收款码图片
1. 将收款码图片保存为:
   - 微信: `wechat_qr.jpg` 或 `wechat_qr.png`
   - 支付宝: `alipay_qr.jpg` 或 `alipay_qr.png`

2. 上传到以下位置之一:
   - GitHub仓库: 上传到 `monetization/images/` 目录
   - 图床服务: 使用 sm.ms / 路过图床 等免费图床

### 3. 修改收款信息

编辑 `payment.html` 文件:

#### 修改收款码图片地址 (第194行和第199行):
```html
<!-- 微信收款码 -->
<img src="./images/wechat_qr.jpg" alt="微信收款码" id="wechatQRImg">

<!-- 支付宝收款码 -->
<img src="./images/alipay_qr.jpg" alt="支付宝收款码" id="alipayQRImg">
```

#### 修改客服微信号 (第218行):
```html
<div>微信号: 你的微信号</div>
```

#### 修改联系邮箱 (第222行):
```html
<div>邮箱: 你的邮箱@example.com</div>
```

## 多产品收款方案

### 方案一: 固定金额收款码
每个产品生成单独的收款码，固定金额

### 方案二: 动态金额收款码
使用支付宝/微信商户版，支持动态金额

### 方案三: 手动确认收款
客户扫码后自行输入金额，你手动确认后发货

## 推荐配置

**最佳实践:**
1. 使用固定金额收款码，减少操作步骤
2. 准备多个金额的收款码: ¥9.9 / ¥19 / ¥29 / ¥39
3. 设置自动回复，确认收款后发送产品链接

## 测试收款

配置完成后:
1. 访问收款页面
2. 用另一个手机扫描测试
3. 转账0.01元测试
4. 确认收款流程正常

## 安全提示

- 不要将收款码图片直接提交到公开仓库
- 使用环境变量或私有配置文件存储敏感信息
- 定期检查收款码是否有效
- 保留收款记录备份
