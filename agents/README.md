# LobsterAI 自动赚钱系统

一个由多个AI智能体组成的自动化赚钱系统，24/7持续运作，无需人工干预。

## 🤖 智能体介绍

### 1. 内容营销机器人 (`content_bot.py`)
- 自动生成各平台营销内容
- 支持小红书、知乎、微信、抖音、微博
- 每天生成8+条高质量内容
- 自动优化标题和标签

### 2. 客户跟进系统 (`customer_bot.py`)
- 自动管理客户关系
- 智能跟进潜在客户
- 自动发送欢迎消息和促销信息
- 客户数据分析和报告

### 3. 数据分析智能体 (`analytics_bot.py`)
- 分析销售数据和趋势
- 优化定价策略
- 评估营销渠道效果
- 生成每日报告

### 4. 联盟营销系统 (`affiliate_bot.py`)
- 管理分销商网络
- 自动生成推广链接
- 计算和发放佣金
- 分销商等级管理

### 5. 知识付费平台 (`knowledge_bot.py`)
- 创建和销售在线课程
- 管理电子书销售
- 预约咨询服务
- 知识产品定价优化

## 🚀 快速开始

### 1. 安装依赖
```bash
cd agents
pip install -r requirements.txt
```

### 2. 配置系统
编辑 `config/scheduler.json` 配置文件，设置：
- 内容生成频率
- 客户跟进策略
- 分销商佣金比例
- 知识产品价格

### 3. 启动系统
```bash
python run_agents.py
```

### 4. 使用任务调度器
```bash
python scheduler.py
```

## 🔧 配置说明

### 收款配置
1. 上传收款码到 `monetization/images/`
2. 编辑 `monetization/config.json` 设置联系方式
3. 测试收款页面

### 内容营销配置
1. 编辑 `config/scheduler.json`
2. 设置发布平台和频率
3. 配置内容主题

### 分销商配置
1. 设置佣金比例
2. 配置分销商等级
3. 设置最低提现金额
