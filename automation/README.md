## 🚀 快速开始

### 1. 配置收款码

访问 `monetization/payment_v2.html`，点击"配置收款信息"按钮，输入：
- 微信收款码图片链接
- 支付宝收款码图片链接
- 客服微信号
- 联系邮箱

### 2. 运行赚钱系统

```bash
python automation/money_maker.py
```

### 3. 管理任务

```bash
python automation/scheduler.py
```
